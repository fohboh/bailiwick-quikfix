Layer 3 FohBoh MGE — Metric Governance Engine · Licensed IP · Deterministic · Version-Controlled fohboh.ai
FohBoh Metric Governance Engine (MGE)
Proprietary licensed technology · Certified System of Record for all platform metrics · Same input always produces same output
KPI Registry
Version-controlled database of all metric formulas. Trust Score weights, QuikFix conditions, revenue calculations — all formulas live here. Every system reads from one canonical source. No independent calculation permitted.
trust_score_v3.2 quikfix_eligibility_v1.4 job_revenue_v2.1
Trust Firewall
Rules-based blocking engine. If data is incomplete or invalid, the metric is blocked — not penalized. Dispatch cannot proceed past a blocked metric. Deterministic, not probabilistic.
IF coi_expired → BLOCK dispatch
IF confidence < threshold → BLOCK quikfix
IF hazard_keyword → BLOCK auto_dispatch
IF location_stale > 5min → BLOCK eta_cert
Certified Metrics API
The single endpoint all subsystems call. Returns certified Trust Score, QuikFix status, revenue figures, and dispatch tier. Response includes mge_version + timestamp. No system calculates metrics independently.
GET /mge/trust-score/:proId
GET /mge/quikfix/:jobId
POST /mge/certify-job
GET /mge/job-revenue/:jobId
MGE OUTPUT → CERTIFIED JOB RECORD (Immutable · Version-Stamped)
certified_job_type certified_flat_rate_usd certified_confidence certified_quikfix_eligible certified_urgency dispatch_tier mge_version mge_timestamp blocking_conditions_triggered[] contractor_trust_score
↓ Certified Job Record → all downstream systems · No raw AI output used beyond this point

Layer 5 Job Execution · Pro PWA + Stripe Connect + S3 + SSE
